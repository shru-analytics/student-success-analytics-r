# Student Success Analytics

Analyzed 500 student records using PCA and K-Means clustering in R.

## Objective
Identify hidden drivers of academic success and segment students for targeted interventions.

## Tools Used
- R
- PCA
- K-Means Clustering
- ggplot2
- corrplot

## Key Findings
- Reduced 8 variables into core performance dimensions
- Identified 4 student segments
- Highlighted at-risk and high-performing groups
